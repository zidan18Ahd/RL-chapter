# Reinforcement Learning Notes

This repository contains structured and research-oriented notes on Reinforcement Learning, with a strong focus on modern deep reinforcement learning methods and their application to Large Language Models.

The goal of this repository is to build a clear conceptual and mathematical understanding of RL algorithms, progressing from foundational methods to advanced policy optimization techniques used in contemporary AI systems.

---

## Repository Structure

RL concepts are organized chapter-wise.

### Foundations

- Markov Decision Process
- Bellman Equations
- Dynamic Programming
- Monte Carlo Methods
- Temporal Difference Learning

### Value-Based Methods

- Q-Learning
- Deep Q Networks (DQN)
- Double DQN
- Dueling Networks

### Policy-Based Methods

- Policy Gradient Theorem
- REINFORCE Algorithm
- Baseline and Advantage Concepts

### Actor-Critic Methods

- Advantage Actor Critic (A2C)
- Asynchronous Advantage Actor Critic (A3C)

### Modern Policy Optimization

- Trust Region Policy Optimization (TRPO)
- Proximal Policy Optimization (PPO)
- Group Relative Policy Optimization (GRPO)

### RL for Language Models

- Reinforcement Learning from Human Feedback (RLHF)
- Reward Modeling
- Reasoning Trajectory Optimization

---

## Learning Philosophy

This repository follows a structured learning flow:

1. Mathematical foundations first
2. Intuition building
3. Algorithmic understanding
4. Implementation perspective
5. Modern research connection

The focus is on understanding why algorithms work, not only how they are implemented.

## Notation Conventions

- s: state  
- a: action  
- r: reward  
- γ: discount factor  
- π: policy  
- V: value function  
- Q: action-value function  
- θ: model parameters
